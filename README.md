# calculadora de descontos
Calculadora de Descontos

Este projeto é uma calculadora de descontos criada para facilitar o cálculo de preços promocionais. O usuário informa o valor original do produto e a porcentagem de desconto, e o sistema retorna automaticamente o valor final a pagar, além do valor economizado.
O objetivo da calculadora é oferecer uma ferramenta simples, rápida e precisa para ajudar consumidores e vendedores a realizarem cálculos de desconto sem complicação.

## Como Clonar

No terminal do VS Code digite:

	git clone https://github.com/Fellipe-07/calculadora-de-descontos.git
	
No GitHub Desktop:

	No menu: File(Arquivo) > Clone a repositor(Clonar repositório) > Selecione a aba URL, no campo URL, cole o link do repositório https://github.com/Fellipe-07/calculadora-de-descontos.git
